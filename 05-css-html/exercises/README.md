# 📝 CSS & HTML Exercises

Practice exercises to master CSS and HTML.

## Exercises

### Beginner Level

#### Exercise 1: Basic Layout
**File**: `exercise-01-basic-layout.html`

Create a page with:
1. A header with navigation
2. A main content area
3. A sidebar
4. A footer
5. Use Flexbox or Grid for layout

---

#### Exercise 2: Card Component
**File**: `exercise-02-card.html`

Create a card component with:
1. Header with title
2. Image
3. Body content
4. Footer with button
5. Hover effects
6. Box shadow

---

#### Exercise 3: Navigation Menu
**File**: `exercise-03-navigation.html`

Build a responsive navigation:
1. Horizontal menu for desktop
2. Hamburger menu for mobile
3. Smooth transitions
4. Active state highlighting
5. Dropdown submenu

---

### Intermediate Level

#### Exercise 4: Responsive Grid

Create a responsive image grid:
1. 3 columns on desktop
2. 2 columns on tablet
3. 1 column on mobile
4. Hover effects on images
5. Lightbox on click (optional)

#### Exercise 5: Form Styling
**File**: `exercise-05-form.html`

Style a registration form:
1. Input fields with labels
2. Focus states
3. Error states
4. Submit button
5. Responsive design
6. Validation styling

#### Exercise 6: Landing Page
**File**: `exercise-06-landing.html`

Build a landing page:
1. Hero section with CTA

4. Footer
5. Responsive design
6. Smooth scrolling

### Advanced Level

#### Exercise 7: CSS Animations

#### Exercise 8: Complex Layout
**File**: `exercise-08-complex.html`

Build a dashboard layout:

#### Exercise 9: CSS Variables & Theming
**File**: `exercise-09-theming.html`

Create a themeable interface:

#### Exercise 10: Clone a Website
**File**: `exercise-10-clone.html`

- Use browser DevTools to inspect and debug
